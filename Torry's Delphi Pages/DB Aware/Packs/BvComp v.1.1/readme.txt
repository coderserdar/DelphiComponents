Installation of library bvComp

To unpack contents of a file bvcomp.zip. Will find subdirectories

bvComp\ - files of library bvComp
bvComp\Demo - the demonstration program.
bvComp\Demo1 - one more demonstration.
bvComp\ENG  - files with the english version of the interface (by default).
bvComp\RUS  - files with the russian version of the interface.


    Compile library bvComp6.dpk (delphi 6) or bvComp5.dpk (delphi 5)
  or bvComp5BC.bpk (C++ Builder 5).
  In a palette of components new page BVComp will open
  set of components.
