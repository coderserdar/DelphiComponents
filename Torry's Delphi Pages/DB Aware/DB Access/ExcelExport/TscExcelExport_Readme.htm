Description

This TscExcelExport component is an advanced, powerful but easy component to export all records of a dataset from Delphi to MS Excel (97, 2000, XP, 2003 or 2007). Many features are provided to change the layout, to add totals, to create groups, ...

Properties

  • Name of worksheet and file

  • Header and footer texts

  • Begin row and column of header, footer, titles (=fieldnames) and data (fieldvalues)

  • Width of columns or autofit

  • Font of header, footer, titles, summary and data (Alignment, WrapText, Orientation, MergeCells)

  • Backgroundcolor and borders of header, titles, summary and data

  • Summaries for numeric or given fields (SUM, MIN, MAX, AVG)

  • Create groups with given fields

  • AutoFilter for titles

  • A lot of other options to customize the result

Events

  • To define the background color and font color, size, name and bold style of each cell

  • For exporting data without using a TDataset

Methods

  • Export to Excel

  • Save worksheet as XLSX (Open XML), XLS, HTML, XML or CSV

  • Show print preview

Component editor

  • The component editor can be used to change some settings on an easy way.
Live templates
  • Live templates for Delphi 2006 (ExcelExportUse and ExcelExportCreate)

Type library / COM

  • This component uses the Office server components from the Borland COM type library.

  • By using compiler directives (in unit scExcelExportConfig.inc) you can override the default type library. The available libraries are Excel97, Excel2000 and ExcelXP (folder Delphi\OCX\Servers).

  • MS Excel should be installed on the PC when using this component.

Delphi / Office

  • This VCL component was tested in Delphi 5 (SP1), Delphi 6, Delphi 7, Delphi (BDS) 2005, Delphi (BDS) 2006 and Turbo Delphi.

  • The TscExcelExport component works with Excel 97, Excel 2000, Excel XP, Excel 2003 and Excel 2007 (bèta) and it has been tested with Dutch and some English Office versions.

  • This component will also give you access to the Excel Application, Workbook and Worksheet objects so you can access all VBA properties and methods.

Demo application

  • A full demo application with 18 examples is supplied to view and test all features of the component (and Excel automation).

Installation

  • Open the run-time package ExcelExport5.dpk (for Delphi5), ExcelExport6.dpk (for Delphi6) ExcelExport7.dpk (for Delphi7), ExcelExport9.dpk (for Delphi 2005), ExcelExport10.dpk (for Delphi 2006 and Turbo Delphi) or ExcelExport11.dpk (for Delphi 2007). "Compile" the package.

  • Open the design-time package dclExcelExportx.dpk and "Compile" and "Install" it. The TscExcelExport component can be found in the tabsheet "SC" of the component palette.

  • When you like to add the component to an existing package, make sure the DCP file dclOffice which can be found in the Delphi/Lib folder is added as "required". When creating a new design-time package, the DCP file designide.dcp (dsnide50.dcp in Delphi5) should be added.

  • The Turbo Delphi (Win32) Explorer edition does not support installing custom components in design time. So the ExcelExportPack10 package can not be installed. The TscExcelExport component should be created in runtime to use it.
