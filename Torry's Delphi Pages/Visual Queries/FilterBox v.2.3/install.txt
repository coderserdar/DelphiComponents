--------------
How to Install
--------------

If you have any previous version of this product, you need
to uninstall it first. 

------------------------
Installation 
------------------------ 

1. Run Delphi.

2. Select the Delphi File->Open menu option and open the file 
\UNITS\psc_delphi.dpk

3. Click Install.

----------------
How to Uninstall
----------------

Please follow these steps when performing uninstall of our 
Delphi components:

1. Call Component|Install Packages dialog of Delphi. 
Then select product package and click "Remove" button. 
After package was removed, click "Ok" button to confirm your changes.

2. Exit Delphi.

3. Delete folder in which component was installed using
Windows Explorer
