CnPack开发组2003年八月份工作总结：

    八月金秋的来临，送走了部分酷暑，迎来了一丝凉风。本月的 CnPack 有了显著的进展，以下是本月总结内容：

一、2003年8月份取得的主要成果：

    1. 专家包的多语版的大部分开发工作已经完成，准备进行简、繁、英的多语测试。
    2. 多语翻译工作已经启动，感谢参与的朋友，欢迎更多朋友参与。
    3. CnPack 新网站正在开发中，预计9月可以初步试运行。
    4. 专家包陆续修正一些小问题。

二、2003年9月初步计划：

    1. 进行专家包的多语以及其他方面的测试工作，修复发现的问题。 
    2. 努力进行帮助文件等的翻译工作。 
    3. 继续完善相关的文档，进行 Code Review。 
    4. 继续进行新网站的设计与开发，预计9月发布试运行版。
