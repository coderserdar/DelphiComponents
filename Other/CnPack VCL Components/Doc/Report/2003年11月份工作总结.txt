CnPack开发组2003年十一月份工作总结：

    十一月份 CnPack 最显著的成果是终于发布了专家包 0.6.7 版，并发布了 CVSTrac 汉化版。本月总结内容如下：

一、2003年11月份取得的主要成果：

    1. 发布 CnPack IDE 专家包 0.6.7 版。该版本修正了大量错误，增加了简繁英的多语支持、对象查看器的粗体显示、对 MSDN.NET 的支持、专家启动时的启用禁用支持等显著功能。
    2. 发布 CVSTrac for Windows 以及 CVSTrac for Linux 汉化版。该错误跟踪工具有利于团队协作开发中的 Bug 报告与错误跟踪处理，配合 CVSNT，是中小团队协作开发的一个不错的解决方案。
    3. 目前的专家开发中，新增 Repository 专家的创建框架，包括菜单列表和模板处理等，并新增一生成 DUnit 测试案例的专家；新增 Hhha 移植的 Explorer 专家，能在 IDE 内部实现资源管理器的功能，支持 IDE 窗体停靠。
    4. CnPack 的 CVSTrac 启用了任务单修改时的 Email 通知功能，更有利于团队中的进度把握和变化跟踪。

二、2003年12月初步计划：

    1. 升级 CnWizards V1.1 框架。
    2. 完善资源管理专家、Explorer 专家、收藏夹专家、书签专家等，并努力进行 FmyExperts、MMC 框架向导和 INI 代码向导等的移植与开发。
    3. 继续补充和重编一些文档。 
    4. 发布专家包 0.6.8 版本。

欢迎对我们提出有价值的意见和建议。
