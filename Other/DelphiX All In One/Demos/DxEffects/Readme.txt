                         -------------------
                         DXEffect SourceCode
                         -------------------

  This is the source code for the DXEffect program that demonstrates
  the water effect, as described in the accompanying article found
  on http://www.gamedev.net: "The Water Effect Explained".

Requirements:

  Borland Delphi 4.0
  DelphiX (found at http://www.ingjapan.ne.jp/hori/index-e.html)
  DirectX 6 or higher (found at the Microsoft website)

The package contains both sourcecode and executable. For copyright
and disclaimer read udelphix1.pas.
