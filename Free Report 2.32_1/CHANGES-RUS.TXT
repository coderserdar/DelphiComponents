Изменения в v2.31:
-----------------------------------------------------------------------------
- Исправлена ошибка при редактировании страницы готового отчета;
- Исправлены утечки ресурсов при печати отчетов, содержащих картинки;
- Исправлены глюки при экспорте отчета;
- Исправлена ошибка, возникающая при удалении компонента TfrToolBar;
- Добавлен компонент TfrPreview для создания окон предварительного просмотра,
  определяемых пользователем;
- Толщина рамки объекта может выражаться дробным числом (0.01 - 10);
- Изменения в коде:
  . переменная Parser переименована в frParser;
  . класс TParser переименован в TfrParser;
  . переменная DataModule переименована в frDataModule.
  . вместо свойств FontName, FontSize и пр. объекта TfrMemoView введено
    свойство Font: TFont.
  . добавлено свойство TfrReport.ModifyPrepared: Boolean для отключения
    возможности редактирования сформированного отчета.
- Значение параметра запроса, созданного в менеджере данных, может быть
  назначено из переменной FR;
- Закладки станиц дизайнера имеют разные заголовки для обычных страниц и для
  страниц с вложенными отчетами;
- Добавлена опция - скрытие заголовков бэндов;
- При вставке нового бэнда он получает название, состоящее из его типа;
- Object inspector сделан в виде окна с изменяемыми размерами;
- Добавлен Add-in TfrRXRichObject (RTF 2.0);
- Добавлен Add-in TfrBarCodeObject для печати штрих-кодов;
- Кросс-отчеты могут иметь переменную высоту строки;
- Изменен алгоритм назначения headers & footers для дата-бэндов. Теперь
  header & footer назначаются ближайшему дата-бэнду. Это позволяет не плодить
  лишних h&f;
- Растягиваемый и разрываемый TfrRichObject;
- Несколько изменено представление полей страницы в дизайнере; показаны
  границы колонок, если отчет многоколоночный;
- Редактор мемо имеет окно с изменяемыми размерами и сплиттером;
- Добавлена возможность визуальной вставки переменной из frVariables
  (в диалогах "Редактор переменных" и "Вставка переменной");
- Добавлена поддержка файлов *.FRP предыдущих версий FR;
- Добавлена поддержка IBObjects (IBO) в ядре репортера (не работает с
  менеджером данных!);
- Добавлена поддержка Interbase Express (IBX), ActiveX Data Objects (ADO)
  в менеджере данных без использования BDE;
- Добавлен стиль линии.
